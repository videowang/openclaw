三机学习环境小抄

目录定位

- 仓库根目录（Mac）：~/Desktop/AI工具集合/OpenClaw_Upstream/openclaw
- 始终在上述目录执行 Git 命令

远程配置

- origin（我的 fork）：https://github.com/videowang/openclaw.git
- upstream（官方仓库）：https://github.com/openclaw/openclaw.git

分支与工作流

- main：主线，仅在 Mac 上合并与推送
- feat/<topic>：实验分支，在 Mac 上开发与提交
- Win11/云端不做 commit/merge/push，只拉取已验证版本

常用命令

```bash
# 查看状态/分支
git status
git branch

# 切换分支
git checkout main
git checkout feat/learning-git
```

提交与推送（在 feat/<topic> 上）

```bash
git add "三机学习环境.md"
git commit -m "docs: add/update 三机学习环境说明"
git push origin feat/learning-git
```

同步上游到本地 main 并推到我的 fork

```bash
git fetch upstream
git checkout main
git merge --ff-only upstream/main
git push origin main
```

部署提示（运行节点只拉取）

```bash
# Win11/WSL2
cd /home/video1/OpenClaw
git fetch origin
git checkout main
git reset --hard origin/main
systemctl restart openclaw

# 云端（备用）
cd /root/OpenClaw
git fetch origin
git checkout main
git reset --hard origin/main
systemctl restart openclaw
```

安全与保密

- 不将 ~/.openclaw/\*、密钥、密码写入仓库
- AI工具集合仓库含敏感运维信息，不推公开仓库

快速检查

```bash
pwd                          # 应为 .../OpenClaw_Upstream/openclaw
git remote -v                # 应有 origin + upstream
git status                   # 工作树状态
```
